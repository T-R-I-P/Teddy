*** このプログラムは現在 Windows 用となっています。Java VM が必要です。

1.実行の仕方

２つの方法があります。

一つは Java のアプレットとしてブラウザ上で実行する方法です。
"teddy.html" ファイルを Internet Explorer 等で開いてください。
この場合には音がなりますが保存はできません。
(動作しない場合には Java をインストールしてください (www.java.com) )

もう一つは アプリケーションとして走らせる方法です。
"teddy.bat" をダブルクリックしてください。
この場合、音はなりませんが、.obj フォーマットで保存できます。
(動作しない場合には Java をインストールして(www.java.com) , 
"java.jar" をダブルクリックしてください。)


2. 注意

このプログラムは研究上のプロトタイプであり、
製品としての利用を前提としたものではありません。
もし動作がおかしくなったらすぐに"Undo"してください。
またそれでも回復しなくなったら、いったん終了して再スタートしてください。
